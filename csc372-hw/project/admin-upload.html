Upload Product Data

Accepted File Formats

  • JSON: Product data formatted as JSON.
  • CSV: Comma-separated values file (columns: name, price, isbn, etc.).
  • Text: Structured plain text file (each product on a new line).

File Format Example

Ensure your file follows this format (JSON example):

[
{
    "product_id": 1,
    "name": "The Handmaid's Tale",
    "image_path": "images/handmaids-tale.jpg",
    "price": 14.99,
    "isbn": "9781250",
    "author": "Margaret Atwood",
    "category": "Fiction",
    "pages": 325,
    "publisher": "RANDOM HOUSE UK",
    "description": "The Handmaid's Tale by Margaret Atwood is a dystopian novel set in a theocratic society
    where women are subjugated and used for reproduction, told through the eyes of a woman named Offred,
    who navigates life under oppressive control and seeks to reclaim her autonomy."
}
]

Instructions

Please ensure the file contains the correct product data format. The JSON format should include all the necessary fields. CSV and text files should follow a similar structure.
