Upload Product Data

Accepted File Formats

  • JSON: Product data formatted as a structured JSON array.
  • CSV: Comma-separated values file where each row represents a product.
  • Text: Key-value structured plain text file where each attribute is listed on a new line.

File Format Example

Ensure your file follows one of these formats:

JSON Example

[
{
    "product_name": "The Handmaid's Tale",
    "description": "76543.",
    "image_url": "images/the-handmaids-tale.jpg",
    "price": 14.99,
    "isbn": "9781250",
    "author": "Margaret Atwood",
    "category_id":1,
    "pages": 325,
    "publisher": "RANDOM HOUSE UK",
    "featured": 0,
    "trending": 1,
    "new_release": 0
  }
]

CSV Example

product_id,name,image_path,price,isbn,author,category,pages,publisher,description
1,The Handmaid's Tale,images/handmaids-tale.jpg,14.99,9781250,Margaret Atwood,Fiction,325,RANDOM HOUSE UK,"The Handmaid's Tale by Margaret Atwood is a dystopian novel set in a theocratic society where women's rights are severely restricted, and they are forced into reproductive roles."

Text Example

product_id: 1
name: The Handmaid's Tale
image_path: images/handmaids-tale.jpg
price: 14.99
isbn: 9781250
author: Margaret Atwood
category: Fiction
pages: 325
publisher: RANDOM HOUSE UK
description: The Handmaid's Tale by Margaret Atwood is a dystopian novel set in a theocratic society where women's rights are severely restricted, and they are forced into reproductive roles.

Instructions

Please ensure that the file you upload follows the correct product data format:

  • JSON format: Include all necessary fields for each product (e.g., product_id, name, price, etc.).
  • CSV and Text formats: Follow a similar structure as the JSON format, with each product's data properly separated by commas (CSV) or new lines (Text).

Each format must adhere to the guidelines provided to ensure smooth data upload and integration into the system.
